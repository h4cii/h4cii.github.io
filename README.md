# h4cii.github.io

Personal CV-style GitHub Pages site for robotics control, reinforcement learning, Sim2Real, and embodied AI.

## Development

```bash
npm install
npm run dev
```

## Video Assets

Place compressed preview videos in `public/media/previews/`. Keep each preview short and privacy-checked before publishing:

- crop sensitive lab background
- remove terminal paths, IP addresses, device IDs, and usernames
- keep full-resolution videos on an external platform or GitHub Release

The public PDF resume lives at `public/resume/hao-yunlai-cv.pdf`.

## Deploy

This repository is configured for GitHub Pages through GitHub Actions. The public site target is:

```text
https://h4cii.github.io
```
